üí¨ Real-time Chat Demo with Pusher

This is a demo of the real-time chat system built with Laravel and Pusher. The chat supports instant messaging between customers and admins with real-time updates.

Features:
  • ‚úÖ Real-time messaging with Pusher
  • ‚úÖ Customer and Admin interfaces
  • ‚úÖ Message history persistence
  • ‚úÖ Connection status indicators
  • ‚úÖ Responsive design
  • ‚úÖ Laravel Echo integration
API Endpoints:
GET /api/pusher-chat/start # Get or create chat for customer POST /api/pusher-chat/messages # Send message GET /api/pusher-chat/messages/{id} # Get chat messages GET /api/pusher-chat/chats # Get all chats (admin only) POST /api/pusher-chat/chats/{id}/reply # Admin reply to chat
Setup Instructions:
  1. Set up Pusher account and get credentials
  2. Add Pusher credentials to .env file
  3. Run migrations: php artisan migrate
  4. Enable broadcasting: Set BROADCAST_DRIVER=pusher
  5. Include Laravel Echo and Pusher JS in your frontend
üîß Environment Configuration

Add these variables to your .env file:

# Broadcasting BROADCAST_DRIVER=pusher # Pusher Configuration PUSHER_APP_ID=your_app_id PUSHER_APP_KEY=your_app_key PUSHER_APP_SECRET=your_app_secret PUSHER_APP_CLUSTER=mt1 # Mix Pusher (for frontend) MIX_PUSHER_APP_KEY="${PUSHER_APP_KEY}" MIX_PUSHER_APP_CLUSTER="${PUSHER_APP_CLUSTER}"
üì± Frontend Integration

Basic JavaScript setup for customers:

// Initialize chat after authentication const chat = new PusherChatCustomer({ pusherKey: '44911da009b5537ffae1', pusherCluster: 'eu', csrfToken: document.querySelector('meta[name="csrf-token"]').content, apiToken: 'user_api_token', baseUrl: 'https://suntop-eg.com', containerId: 'chat-container' });

For Laravel Blade templates:

<script> document.addEventListener('DOMContentLoaded', function() { const chat = new PusherChatCustomer({ pusherKey: '44911da009b5537ffae1', pusherCluster: 'eu', csrfToken: '{{ csrf_token() }}', apiToken: '{{ auth()->user()->createToken("chat")->plainTextToken }}', baseUrl: '{{ url("/") }}', containerId: 'chat-widget' }); }); </script>
