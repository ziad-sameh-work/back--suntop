🧪 اختبار Product API Endpoints

📋 التعليمات:

  • اضغط على "Test API" لاختبار الـ endpoint
  • اضغط على "Copy cURL" لنسخ أمر cURL
  • تأكد من وجود back_color في النتائج
  • جميع الـ endpoints تُرجع JSON response

🔗 Public API Endpoints

📦 جميع المنتجات

يُرجع قائمة بجميع المنتجات مع pagination

GET
/api/products
اضغط "Test API" لعرض النتائج

⭐ المنتجات المميزة

يُرجع المنتجات المميزة فقط

GET
/api/products/featured
اضغط "Test API" لعرض النتائج

🔍 منتج واحد

يُرجع تفاصيل منتج واحد (تفاصيل كاملة)

GET
/api/products/1
اضغط "Test API" لعرض النتائج

🔎 منتجات مع فلاتر

يُرجع منتجات مع فلاتر (category, search, featured)

GET
/api/products?featured=1&limit=5
اضغط "Test API" لعرض النتائج

🔐 Admin API Endpoints (تحتاج authentication)

👨‍💼 منتجات الإدارة

يُرجع منتجات للإدارة مع تفاصيل إضافية (يحتاج token)

GET
/api/admin/products
اضغط "Test API" لعرض النتائج (قد يفشل بدون authentication)

📊 back_color Field Verification

✅ ما يجب أن تجده في النتائج:

  • "back_color": "#FFFFFF" أو أي لون hex صحيح
  • إذا كان "back_color": null فهناك مشكلة
  • إذا لم يظهر back_color نهائياً فهناك مشكلة في الـ Resource
